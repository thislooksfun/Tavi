  • fromPos(_:)

    Get a NoteType from its position

    The return values are as follows: \ 0: .Start \ 1: .Pass \ 2: .Fail \ 3: .Cancel \

    If pos < 0 or pos > 3, it will return .None

    Parameter

    Parameter pos: The position, in the interval [0,3]

    Returns

    .Start, .Pass, .Fail, or .Cancel if pos is valid, otherwise .None
